3. Se definen las variables independientes: frecuencia, ngulo, longitud, velocidad y espesor. La variable dependiente es la sexta del archivo, siendo la presin. Luego se entrena el modelo de regresin lineal mltiple para que las 5 variables independientes intenten predecir la dependiente.

Se imprime un resumen del modelo, as como los 'valores p' con el atributo pvalues.

In[7]:
import statsmodels.api as sm
x=data[['frecuencia','angulo','longitud','velocidad','espesor']]
y=data['presion']
x=sm.add_constant(x)

x_train,x_test,y_train,y_test=train_test_split(x,y,test_size=0.3,random_state=42)
modelo=sm.OLS(y_train,x_train).fit()

print(modelo.summary())
print("p values: ",modelo.pvalues)
